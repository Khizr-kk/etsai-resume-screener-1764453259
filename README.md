# FreshPath AI

Auto-generated by ETS-AI 10x.

## Idea Summary
AI-powered resume screener for Indian fresh software engineers, providing scores and missing skills based on job descriptions.

## Problem Statement
Indian fresh software engineers struggle to effectively tailor their resumes for job applications, leading to low success rates and difficulty in identifying skill gaps.

## Target Persona
Ideal customer

## MVP Features
- User authentication and profile management
- Resume upload (PDF, DOCX) and parsing
- Job description input (text field)
- AI-powered resume-job description matching algorithm (scoring)
- Identification and display of missing skills
- Basic dashboard to view past screenings

## Tech Stack
Python (Flask/Django) for backend and ML, React/Vue.js for frontend, PostgreSQL for database, AWS/GCP for cloud infrastructure.
